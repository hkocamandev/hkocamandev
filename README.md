## About

Senior software engineer with 8+ years in Java backend, distributed systems, and security platforms — currently moving into AI/LLM engineering. Most of my recent work sits at the intersection of the two: event-driven services, vector search, and ML inference wired into real backends rather than notebooks.

```yaml
location:   İzmir, Türkiye
focus:      Java / Spring backend  +  AI engineering (RAG, agents, evals)
education:  M.Sc. Computer Engineering @ METU (ML / Deep Learning, LSTM thesis)
            B.Sc. Computer Engineering @ Turkish Air Force Academy (ranked 1st)
```

- 🔭 Building out a set of LLM-focused projects: an eval-driven RAG system, LLM observability tooling, and a production-constrained agent.
- 🌱 Spending time on retrieval, embeddings, evals, and the Model Context Protocol (MCP).
- 📜 Working towards the AWS Certified AI Practitioner (AIF-C01).
- 🧠 Background in distributed systems, event-driven architecture, and identity (SAML, SSO, OAuth2).

## Projects

| Project | What it does | Stack |
|---|---|---|
| **[iot-intrusion-detection](https://github.com/hkocamandev/iot-intrusion-detection)** | Kafka streaming pipeline for near real-time intrusion detection on RT-IoT2022 traffic. Runs rule-based and ML-based (XGBoost over FastAPI) detection as separate consumer groups, with config-driven rules and retry / dead-letter handling. | `Java 21` `Spring Boot` `Kafka` `PostgreSQL` `Python` `FastAPI` `XGBoost` `Testcontainers` |
| **[collab-notes](https://github.com/hkocamandev/collab-notes)** | Notion-style collaborative editor: CRDT sync, live presence, version history, and local semantic search over your notes (no external API) — also exposed over MCP. | `React` `TypeScript` `Yjs` `Tiptap` `Express` `Prisma` `Transformers.js` `MCP` |
| **[ShardTicket](https://github.com/hkocamandev/ShardTicket)** | A benchmark comparing a sharded MongoDB cluster with ACID transactions against a no-transaction replica set under concurrent load — to show why one oversells tickets and the other doesn't. In-browser k6 load testing, Redis toggles, Prometheus + Grafana. | `Node.js` `Express` `MongoDB` `Redis` `React` `k6` `Prometheus` `Grafana` |
| **[sharp-bunny](https://github.com/hkocamandev/sharp-bunny)** | Image processing pipeline on RabbitMQ — async resizing, automatic retries, dead-letter handling, watermarking, and a real-time dashboard. | `Node.js` `RabbitMQ` `Sharp` `Express` `Socket.IO` |
